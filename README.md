# Sistema POS de venta de Zetas

> [!NOTE] Este proyecto esta enfocado en el desarrollo y aprendizaje de nuevas tecnologias

Sistema de ventas con tematica de zetas.

Este projecto tendra las funcionalidades de:

- Carrito y ventas
- Inventario
- Reportes
- Roles de Usuario
- Tikets
- Catalogo

Se implemnetara una arquitectura de monorepos con pnpm, ademas del uso de Typescript, React y node

Tecnologias:

|    Root    | Front-end | Back-end |
| :--------: | :-------: | :------: |
| typescript |   React   |   Node   |
